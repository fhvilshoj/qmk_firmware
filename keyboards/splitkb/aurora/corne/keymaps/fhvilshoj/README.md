### FUNCTION KEYS AND MOUSE

```
╭────┬────┬────┬────┬────┬────╮             ╭────┬────┬────┬─────┬─────┬─────╮
│ F1 │ F2 │ F3 │ F4 │ F5 │ F6 │             │ F7 │ F8 │ F9 │ F10 │ F11 │ F12 │
├────┼────┼────┼────┼────┼────┤             ├────┼────┼────┼─────┼─────┼─────┤
│    │    │ Vo │ V- │ V+ │    │             │ ←  │ ↓  │ ↑  │  →  │     │     │
├────┼────┼────┼────┼────┼────┤             ├────┼────┼────┼─────┼─────┼─────┤
│ BT │    │    │ B+ │ B+ │    │             │    │ LC │ RC │     │     │     │
╰────┴────┴────┴────┴────┴────╯             ╰────┴────┴────┴─────┴─────┴─────╯
                   ╭─────┬────┬─────╮ ╭─────┬─────┬─────╮
                   │ CMD │ -- │ ENT │ │ SPC │ --- │ ALT │
                   ╰─────┴────┴─────╯ ╰─────┴─────┴─────╯
```

`BT`: Put keyboard into bootloader mode.  
`Vo`: Volume off  
`V-`: Volume down  
`V+`: Volume up  
`B-`: Brightness down  
`B+`: Brightness up  
`LC`: Left mouse click  
`RC`: Right mouse click
